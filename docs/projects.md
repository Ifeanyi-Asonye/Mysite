
# Projects

### 1. Cloud Infrastructure Automation
**Description:** Designed and deployed production-grade infrastructure for an enterprise SaaS platform, with a focus on high availability, cost optimization, and security compliance.

**Key Features:**
- Infrastructure as Code (IaC) with Terraform and AWS CloudFormation
- CI/CD pipeline for automated provisioning and drift detection in GitHub Actions
- Auto-scaling groups, VPC segmentation, and multi-AZ failover
- Logging and alerts via CloudWatch, Datadog, and PagerDuty
